# Тесты

- `npm run test:e2e:heavy` — тяжёлые сценарии Мымры: D3-граф, tooltip, экспорт, импорт и слияние. Запускайте перед крупными изменениями графа, импорта или перед релизом.

Лёгких browser-тестов пока нет: после их появления для них будет отдельная быстрая команда.

Тяжёлые тесты используют системный Edge через Playwright и поднимают временный Vite-сервер на порту `5177`.
